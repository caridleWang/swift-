Swift是一种高性能的编程语言，具有简单干净的语法，可以与C和OC进行无缝对接，并且默认是内存安全的。 在Swift中可以看到，C、OC和许多其他语言的影子，但其并非C的派生语言。Swift,作为一种强大的开源编程语言，一定可以让我们开发出更为优秀的APP。OK，下边是总结的博客、代码和教程等资源【基础资源】。
